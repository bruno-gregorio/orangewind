Orangewind

Canonical's Vanilla Framework, re-implemented as a Tailwind CSS v4 plugin — and these docs are built with it.

Orangewind brings Vanilla's component set to Tailwind projects as a set of ow- classes, reusing Tailwind's utilities, theme and dark variant rather than duplicating them.

Get started Browse components GitHub

Why Orangewind

A Tailwind v4 plugin

Vanilla Framework re-implemented as a Tailwind plugin — it reuses Tailwind's native utilities instead of duplicating them.

Vanilla parity

Faithful ports of Vanilla's components — buttons, forms, navigation, heroes, suru and more — under a single ow- prefix.

Class-based markup

Plain HTML with ow- component classes. No required JavaScript, single-dash kebab-case names, composes with raw Tailwind.

Automatic dark mode

Theming runs through CSS variables flipped by Tailwind's dark variant, so the framework follows the OS theme with no toggle.
