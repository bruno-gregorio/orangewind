A media object places an image alongside a block of text, such as an event, a person or a resource. Pair an ow-media-object-image with an ow-media-object-details block; this page also covers the image helpers (ow-image-container-*, ow-image-bordered and ow-image-shadowed) used to frame imagery elsewhere.

Default

The image sits to the left of a title, a line of content and an optional meta list. Meta items can use the typed classes ow-media-object-meta-item-date, -location and -venue to show an icon, or the plain ow-media-object-meta-item for none.

NVIDIA GTC 2019

Ubuntu powered everything from autonomous cars to inference at the edge.

  • Date: 18 - 21 March 2019
  • Venue: San Jose McEnery Convention Center
Markup
<div class="ow-media-object">
  <img class="ow-media-object-image" src="/path/to/icon.svg" alt="" />
  <div class="ow-media-object-details">
    <h3 class="ow-media-object-title">
      <a href="#nvidia-gtc">NVIDIA GTC 2019</a>
    </h3>
    <p class="ow-media-object-content">
      Ubuntu powered everything from autonomous cars to inference at the edge.
    </p>
    <ul class="ow-media-object-meta-list">
      <li class="ow-media-object-meta-item-date">
        <span class="sr-only">Date: </span>
        <span>18 - 21 March 2019</span>
      </li>
      <li class="ow-media-object-meta-item-venue">
        <span class="sr-only">Venue: </span>
        <span>San Jose McEnery Convention Center</span>
      </li>
    </ul>
  </div>
</div>

Circular image

Add ow-is-round to the image to crop it to a circle, which suits avatars and profiles.

Image frames

Frame a standalone image with ow-image-bordered for a hairline border or ow-image-shadowed for a soft shadow.

Markup
<img class="ow-image-bordered" src="/path/to/photo.png" alt="" width="640" height="360" />
<img class="ow-image-shadowed" src="/path/to/photo.png" alt="" width="640" height="360" />

Aspect ratio containers

Wrap an ow-image-container-img in a ratio container such as ow-image-container-16-9, ow-image-container-3-2, -2-3, -cinematic or -square to lock its proportions. Add ow-is-highlighted for a placeholder background, or ow-is-cover to crop the image to fill the frame.

Markup
<div class="ow-image-container-16-9 ow-is-cover">
  <img class="ow-image-container-img" src="/path/to/landscape.jpeg" alt="" />
</div>
<div class="ow-image-container-square ow-is-highlighted">
  <img class="ow-image-container-img" src="/path/to/product.png" alt="" />
</div>

Media spacing

ow-media-container adds a small amount of top padding, useful for separating an image from a preceding rule or heading.

Markup
<hr class="ow-rule" />
<div class="ow-media-container">
  <img src="/path/to/photo.png" alt="" width="640" height="360" />
</div>
