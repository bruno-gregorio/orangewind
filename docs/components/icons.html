Icons are single-glyph elements rendered as a CSS mask, so they take their colour from the current text colour and scale with width and height utilities. Apply an ow-icon-* class to an empty <i> (or <span>); include a text label as its content for assistive technology, which is visually hidden by the icon styles.

Default icons

Each icon has its own ow-icon-* class. At their default size they match the surrounding text, making them easy to drop inline.

information success warning error help
Markup
<i class="ow-icon-information">information</i>
<i class="ow-icon-success">success</i>
<i class="ow-icon-warning">warning</i>
<i class="ow-icon-error">error</i>
<i class="ow-icon-help">help</i>

Sizing

Icons render at 1rem by default. Set explicit width and height utilities such as h-6 w-6 or h-10 w-10 to scale them up; keep the two equal so the icon stays square.

settings settings settings
Markup
<i class="ow-icon-settings h-4 w-4">settings</i>
<i class="ow-icon-settings h-6 w-6">settings</i>
<i class="ow-icon-settings h-10 w-10">settings</i>

Colour

Most icons inherit the current text colour, so a text colour utility recolours them. The semantic status icons — ow-icon-error, ow-icon-warning and ow-icon-success — carry their own meaningful colour and ignore the text colour.

starred starred starred
Markup
<i class="ow-icon-starred text-ow-brand">starred</i>
<i class="ow-icon-starred text-ow-positive">starred</i>
<i class="ow-icon-starred text-ow-information">starred</i>

Icons in context

Because an icon inherits colour and aligns to the middle of the line, it sits naturally inside buttons, links and headings. Inside a button, combine it with ow-has-icon.

security Security

Markup
<button class="ow-btn-positive ow-has-icon">
  <i class="ow-icon-plus"></i>
  <span>Add item</span>
</button>
<h3 class="ow-heading-3">
  <i class="ow-icon-security"></i>
  Security
</h3>

Status icons

Small status icons such as ow-icon-status-succeeded-small and ow-icon-status-waiting-small are designed to annotate rows and list items at a glance.

succeeded in progress waiting failed
Markup
<i class="ow-icon-status-succeeded-small">succeeded</i>
<i class="ow-icon-status-in-progress-small">in progress</i>
<i class="ow-icon-status-waiting-small">waiting</i>
<i class="ow-icon-status-failed-small">failed</i>
