Feature blocks are the marketing-page building blocks: pricing tables, resource lists, logo walls, calls to action, data spotlights and newsletter sign-ups. Each composes Orangewind primitives — sections, rules, the ow-cta-block and image containers — into a self-contained region. This page shows one representative example of each block.

Pricing block

The ow-pricing-block grid lays tiers side by side, each a subgrid so their sections align row by row. Variants ow-pricing-block-25-75 and ow-pricing-block-50-50 centre three or two tiers. List features with ow-list-divided and ow-is-ticked / ow-is-crossed markers.

Resources block

An ow-resources-block pairs a media column with a content column, splitting 25/75 on large screens. Stack several behind a section header and separate them with ow-rule-muted.

Logo section

An ow-logo-section arranges partner logos in a wrapping flex row. Each ow-logo-section-logo keeps a consistent height. Introduce it with a header and a fixed-width rule.

CTA section

A call-to-action section sets a headline (often with an inline link) on an emphasis strip. Pair it with an ow-cta-block when you need buttons. The block aligns its actions on a shared baseline and wraps as needed.

Newsletter signup

A newsletter block introduces a stacked sign-up form. Use ow-form ow-form-stacked for the fields and an ow-checkbox for the opt-in. The 25/75 layout puts the heading beside the form on large screens.

Newsletter signup

Get the latest Ubuntu news and updates in your inbox.

Markup
<div class="ow-section">
  <hr class="ow-rule ow-is-fixed-width" />
  <div class="grid gap-6 pt-4 lg:grid-cols-[1fr_3fr]">
    <h2 class="ow-heading-2">Newsletter signup</h2>
    <div class="grid gap-4">
      <p>Get the latest Ubuntu news and updates in your inbox.</p>
      <form action="#subscribe" method="post" class="ow-form ow-form-stacked">
        <label class="ow-is-required" for="email">Work email:</label>
        <input required id="email" name="email" type="email" placeholder="you@example.com" />
        <div class="ow-checkbox">
          <input required type="checkbox" id="optin" class="ow-checkbox-input" />
          <label for="optin" class="ow-checkbox-label">I agree to receive product news.</label>
        </div>
        <button type="submit" class="ow-btn-positive">Sign up</button>
      </form>
    </div>
  </div>
</div>
