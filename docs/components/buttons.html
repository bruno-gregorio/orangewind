Documentation

Buttons are clickable elements used to perform an action. You can apply button classes to <button> and <a> elements. Do not use multiple button classes on a single element.

Default

A default button indicates a positive action that isn't necessarily the main call to action.

Markup
<button class="ow-btn">Default</button>

Positive

A positive button indicates the main, affirmative call to action.

Markup
<button class="ow-btn-positive">Positive</button>

Base

A base button discreetly indicates a secondary action.

Markup
<button class="ow-btn-base">Base</button>

Negative

A negative button indicates a destructive or permanent action.

Markup
<button class="ow-btn-negative">Negative</button>

Link

A link button looks visually identical to a link but behaves as a button.

Markup
<button class="ow-btn-link">Link</button>

Small and dense

Add ow-is-small for small text contexts and ow-is-dense for reduced vertical padding. They can be combined.

Markup
<button class="ow-btn ow-is-small">Small</button>
<button class="ow-btn ow-is-dense">Dense</button>

With an icon

Add ow-has-icon to align an icon alongside the label, or use it alone for an icon-only button (always provide an accessible label).

Markup
<button class="ow-btn-positive ow-has-icon">
  <i class="ow-icon-plus"></i>
  <span>Add</span>
</button>
