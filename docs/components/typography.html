Typography utilities set the type scale for the system. Apply a heading class to any element to give it heading styling without dictating its semantic level, so document structure and visual size stay independent.

Headings

Six heading classes, ow-heading-1 through ow-heading-6, descend in size and weight. Use the class that matches the visual hierarchy and choose the heading element that matches the document outline.

Markup
<h2 class="ow-heading-1">Heading 1</h2>
<h2 class="ow-heading-2">Heading 2</h2>
<h3 class="ow-heading-3">Heading 3</h3>
<h4 class="ow-heading-4">Heading 4</h4>
<h5 class="ow-heading-5">Heading 5</h5>
<h6 class="ow-heading-6">Heading 6</h6>

Display heading

The ow-display-heading class is larger and lighter than the numbered headings. Reserve it for the single most prominent title on a page, such as a hero or section opener.

Display heading

Markup
<h1 class="ow-display-heading">Display heading</h1>

Muted heading

A muted heading sets a small-caps eyebrow above a title. Use ow-muted-heading to label a section without competing with the heading beneath it.

Latest news

Canonical expands AI portfolio

Markup
<span class="ow-muted-heading">Latest news</span>
<h2 class="ow-heading-2">Canonical expands AI portfolio</h2>

Small caps

The ow-text-small-caps utility renders a block of text in small caps with old-style numerals and slightly looser letter spacing.

Small caps utility text block

Markup
<p class="ow-text-small-caps">Small caps utility text block</p>

Heading with an icon

Pair a heading with an image using ow-heading-icon-header. Add ow-is-stacked to place the image above the title, or wrap the block in ow-heading-icon-small for a smaller image.

LXD — system containers

Markup
<div class="ow-heading-icon-header">
  <img
    class="ow-heading-icon-img"
    src="https://assets.ubuntu.com/v1/425efe3a-lxd.svg"
    alt=""
  />
  <h3 class="ow-heading-icon-title ow-heading-3">LXD — system containers</h3>
</div>
