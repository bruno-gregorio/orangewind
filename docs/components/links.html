Documentation

Read the Vanilla documentation or jump straight to the links pattern.

Anchor elements are styled by default. Additional classes adjust their emphasis or adapt them to dark backgrounds.

Default

A standard link needs no class — the base styles apply to every anchor.

A standard link
Markup
<a href="#">A standard link</a>

Soft

A soft link uses the text colour until hovered, for links that should sit quietly within a block of text.

A soft link
Markup
<a class="ow-link-soft" href="#">A soft link</a>

Inverted

Use an inverted link on dark backgrounds so it keeps sufficient contrast.

An inverted link
Markup
<div class="ow-strip-dark">
  <a class="ow-link-inverted" href="#">An inverted link</a>
</div>

Skip link

A skip link lets keyboard users jump past repeated navigation to the main content. It stays hidden until focused — tab to the preview to reveal it.

Skip to main content
Markup
<a class="ow-link-skip" href="#main-content">Skip to main content</a>
